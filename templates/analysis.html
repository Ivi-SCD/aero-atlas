Construção da Ferramenta

Nossa ferramenta foi desenvolvida utilizando Python no backend e JavaScript no frontend. Utilizamos a biblioteca Flask para criar nossa API RESTful e Leaflet.js para renderizar os mapas interativos.

Cálculos Utilizados

NDVI (Índice de Vegetação por Diferença Normalizada)

NDVI = (NIR - RED) / (NIR + RED)

Onde NIR é a banda do infravermelho próximo e RED é a banda do vermelho.

EVI (Índice de Vegetação Melhorado)

EVI = 2.5 * ((NIR - RED) / (NIR + 6 * RED - 7.5 * BLUE + 1))

Onde BLUE é a banda do azul.

SAVI (Índice de Vegetação Ajustado ao Solo)

SAVI = ((NIR - RED) / (NIR + RED + L)) * (1 + L)

Onde L é um fator de ajuste do solo, geralmente 0.5.

NDWI (Índice de Água por Diferença Normalizada)

NDWI = (GREEN - NIR) / (GREEN + NIR)

Onde GREEN é a banda do verde.

MNDWI (Índice de Água por Diferença Normalizada Modificado)

MNDWI = (GREEN - SWIR) / (GREEN + SWIR)

Onde SWIR é a banda do infravermelho de ondas curtas.

AWEI (Índice de Extração de Água Automatizado)

AWEI = 4 * (GREEN - SWIR) - (0.25 * NIR + 2.75 * SWIR)

Curiosidades Científicas

  • O satélite Landsat 8 orbita a Terra a cada 99 minutos, completando 14 órbitas por dia.
  • Cada cena Landsat cobre uma área de aproximadamente 185 km x 180 km.
  • O programa Landsat é o programa de observação da Terra mais longo da história, com mais de 40 anos de dados contínuos.
  • As imagens Landsat são usadas em diversos campos, desde agricultura até estudos de mudanças climáticas.
